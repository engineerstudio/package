# 包网项目开源
这里是包网的后台完整项目, 具体包含以下模块
1> 会员模块,包含会员管理、代理管理、分组管理等
2> 资金模块,包含充值订单、出款订单、支付与代付渠道
3> 游戏模块,包含游戏相关的订单,游戏报表等
4> 活动模块,包含常见的十种优惠活动的创建、订单等
5> 内容模块,包含页面一些文字类的管理等
6> 系统模块,包含权限账户、站点一些简单的配置等
